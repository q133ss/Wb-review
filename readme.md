# wbReviewBot

Простой веб-сервис для ответов на вопросы и отзывы покупателей маркетплейса с помощью ИИ.

## Требования заказчика
- Получать вопросы и отзывы через API WB.
- Отправлять текст в ИИ (промпт предоставляется заказчиком).
- Показывать ответ в интерфейсе, давать возможность редактировать.
- После подтверждения менеджером отправлять ответ в WB через API.
- Хранить историю обращений и ответов.
- Автоответы для 4 и 5 звезд.
- Промпт ИИ должен редактироваться в настройках без изменения кода.
- Технологии: Python (FastAPI) или Node.js, простая БД, web-интерфейс.

## Планируемый функционал сервиса
- Импорт новых вопросов/отзывов из WB.
- Генерация ответа через ИИ по настраиваемому промпту.
- Админ-интерфейс: текст покупателя, ответ ИИ, поле редактирования, кнопка отправки.
- Ручное подтверждение ответа менеджером.
- История диалогов и статусы отправки.

## Технологии (план)
- FastAPI
- Простая БД (SQLite/PostgreSQL)
- Минимальный web-интерфейс для админки
